# EcoHabit - Project Brief

## Project Overview
**EcoHabit** is a gamified waste management and recycling application designed to encourage users to adopt sustainable habits through psychological reward systems and behavioral change techniques.

## Mission & Vision
- **Mission**: Transform waste management from a chore into an engaging, rewarding habit through gamification and psychological reinforcement
- **Vision**: Create a self-happiness driven ecosystem where environmental action becomes intrinsically motivating

## Target Audience
- **Primary**: Eco-conscious individuals aged 18-40 in Sri Lanka
- **Secondary**: Students and young professionals interested in sustainable living
- **Tertiary**: Communities and organizations promoting green lifestyles

## Geographic Focus
**Sri Lanka** - addressing specific local waste management challenges:
- 7,000 metric tons daily waste generation
- 65% disposed via open dumping
- Limited formal recycling infrastructure
- High smartphone penetration (75-85% in urban areas)

## Core Problem Statement
Sri Lanka faces a critical waste management crisis:
- Only 50% of generated waste is collected
- 65% goes to open dumps causing environmental degradation
- Limited public awareness of proper waste segregation
- Insufficient recycling infrastructure and education

## Key Objectives
1. **Behavioral Change**: Convert waste management from external obligation to internal motivation
2. **Habit Formation**: Create lasting recycling habits through psychological reward loops
3. **Education**: Provide engaging, story-driven environmental education
4. **Community Impact**: Enable collective action and social validation
5. **Measurable Impact**: Track and visualize environmental contributions

## Success Metrics
- **User Engagement**: Daily Active Users (DAU), session duration, retention rates
- **Behavioral Impact**: Average streak length, actions logged per user
- **Environmental Impact**: Waste diverted from landfills, CO2 emissions prevented
- **Community Growth**: User acquisition, social sharing, friend invitations

## Technology Stack Overview
- **Frontend**: Next.js (web), Expo (mobile)
- **Backend**: Firebase (serverless)
- **AI**: Google Gemini API for personalized coaching
- **3D**: Three.js for gamified visualizations
- **Testing**: Playwright (E2E), Jest + React Testing Library (unit)
- **Architecture**: Monorepo with Turborepo

## Development Approach
- **Spec-Driven Development**: AI-assisted implementation based on comprehensive documentation
- **Behavioral Psychology First**: Design decisions based on habit formation research
- **Cost-Optimized**: Firebase free tier for MVP, scalable architecture
- **Localized**: Multi-language support (English, Sinhala, Tamil)
- **Mobile-First**: PWA with native mobile app for maximum reach

## Core Value Proposition
Transform environmental action from external pressure to internal satisfaction through:
- **Instant Gratification**: Immediate rewards and feedback
- **Progress Visualization**: Tangible impact measurement
- **Social Recognition**: Community validation and competition
- **Educational Journey**: Story-driven learning experiences
- **Habit Reinforcement**: Psychological reward loops
